# Folp-Extension
Clone the project into a folder.

cd server → npm install → create .env from .env.example and fill API keys.

npm start to run the backend (default port 5000).

Load extension in Chrome: go to chrome://extensions → enable Developer mode → "Load unpacked" → select the extension/ folder.

On any page: select text or right-click an image → choose Search with Folp → results appear as a transparent overlay.
