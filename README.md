Folp is a Chrome Extension + Node.js backend that helps identify people from faces, names, or selected text.
It works like Bixby Vision or Google Lens, fetching public info across the web.

Features

Chrome Extension with content + background scripts.

Backend powered by Express.js with API integrations.

Face, name, and image recognition support.

Cache layer for fast repeated lookups.

Circuit breaker to handle API failures gracefully.

Secure API key management with .env.

Context menu support for quick searches.

Overlay UI for displaying results on any website.

Lightweight design, easy to use.

Open-source and extendable.

Setup

Clone repo and run npm install in backend.

Add API keys in .env.

Load extension/ as unpacked in Chrome.
